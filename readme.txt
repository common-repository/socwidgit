== Description ==

With this plugin you can easy place some Social Like buttons to sidebar.
Current release supports vk.com (vkontakte.ru) Like, Facebook Like, Twitter Like and Google Plus (+1)

== Installation ==

* Go to Plugins section in your admin panel
* Find and Activate SocWidgIt! plugin
* Go to Widgets section, find SocWidgIt! widget and place it to sidebar
* Make your settings to bind social accounts, using widget configuration.
* If you use cache plugins, clear cache. You will see result.

== Frequently Asked Questions ==

= Social buttons are not working proper =

Please check your IDs and settings in social networks.
